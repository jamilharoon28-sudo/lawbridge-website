LAWBRIDGE NETLIFY DROP-READY WEBSITE

This folder is ready to drag directly into Netlify Drop.

How to use:
1. Unzip this package.
2. Open the folder named LawBridge_Netlify_Drop_Ready.
3. Drag the CONTENTS or the full folder into Netlify Drop.
4. Netlify will create a live preview link.

No npm install is needed.
No build command is needed.
No GitHub setup is needed for testing.

Included:
- index.html
- favicon.svg
- _redirects

The website is a complete static prototype with:
- Home
- Curriculum
- Students
- Mentors
- Workshops / Programmes
- Webinars
- Resources
- Blog
- About
- Contact / Partnership Hub
- Testable forms
- Local prototype submissions
- CSV export
- Responsive mobile navigation
- Seamless section transitions

Prototype forms save submissions inside the browser only. Production forms should connect to HubSpot, Airtable, Netlify Forms, WordPress forms, or a backend.
