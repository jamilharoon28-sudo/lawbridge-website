LAWBRIDGE CMS ADMIN READY

Upload all files in this folder to your GitHub repository.

Includes:
- index.html
- admin/index.html
- admin/config.yml
- content/site.json

After uploading:
1. Netlify will redeploy.
2. Go to /admin.
3. Accept your Netlify Identity invite.
4. Edit Main Website Text.
5. Publish.

Note: This starter CMS creates the dashboard and editable content file.
The next step is fully wiring every section/card to content/site.json if you want every page element editable from the dashboard.
